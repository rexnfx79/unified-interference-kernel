# Connes Spectral Triple (Curated Summary)

> **Primary source:** Connes, *Noncommutative Geometry* (1994); Connes "An essay on the Riemann Hypothesis" (2016).  
> **Wiki concepts:** [[hilbert-polya-conjecture]], [[riemann-zeta-function]], [[random-matrix-theory]]

## Program (conjectural framework)

Connes proposed that the critical zeros of \(\zeta(s)\) arise as an **absorption spectrum** in a noncommutative geometric (NC) model built from:

- A **spectral triple** \((\mathcal{A}, \mathcal{H}, D)\): algebra, Hilbert space, Dirac operator
- Adele / idele class space as geometric backdrop
- Trace formulas linking spectral side to arithmetic side

This is a modern incarnation of the **Hilbert–Polya** idea: find an operator whose spectrum encodes zeros.

## Relation to Montgomery / GUE

Connes' program **motivates** an operator picture compatible with GUE-like zero statistics, but it **does not prove** Montgomery pair correlation or full GUE universality. Numerical GUE match remains Odlyzko/Montgomery territory ([[montgomery-pair-correlation]]).
