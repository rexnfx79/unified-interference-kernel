## Title

Unified Interference Kernel: Three Projection Regimes for Fermion Flavor

## Abstract (Condensed)

Framework organizing Yukawa couplings via envelope suppression and phase interference. Single kernel **form** applied across fermion sectors; sectors correspond to distinct projection regimes: envelope-dominated (quarks), phase-sensitive (leptons), metric-dominated (neutrinos).

Primary result: quark sector CKM–\(m_c\) Pareto trade-off. Secondary: lepton masses and PMNS angles reproduced with survivor rates 60% and 45%.

## Kernel

\[
Y_{ij} = \exp(-d_{ij}^2/2\sigma^2) \times [1 + \varepsilon \exp(i\Phi_{ij})]
\]

\[
\Phi_{ij} = \alpha + k(x_i+x_j)/2 + \eta(x_i-x_j)
\]

"Universal" = single functional form, NOT parameter value universality or UV origin.
