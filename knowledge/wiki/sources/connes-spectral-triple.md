# Connes Spectral Triple

**Tier 5.1 ingest** — noncommutative geometry (NC) program toward Hilbert–Polya; independent of flavor phenomenology.

## Program sketch

Connes proposes critical zeros of \(\zeta(s)\) as an **absorption spectrum** built from:

- **Spectral triple** \((\mathcal{A}, \mathcal{H}, D)\): algebra, Hilbert space, Dirac operator
- **Adele / idele class space** as geometric backdrop
- **Trace formula** whose spectral side encodes zeros and geometric side encodes primes (Weil explicit formula as template)

## Relation to Montgomery / Selberg

- **Montgomery:** statistical spacing law for zeros (conditional) — does not construct \(D\)
- **Selberg:** proved geodesic↔spectrum on hyperbolic surfaces — different space, same **template**
- **Connes:** aims to realize Riemann’s side with NC trace technology

## Assumed-true discipline

Many researchers **expect** an operator-like object behind zeros; Connes gives the most developed **framework**. That is not evidence for any **specific** SM parameter relation.
