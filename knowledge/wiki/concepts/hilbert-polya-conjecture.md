# Hilbert–Polya Conjecture

## Statement

There exists a **self-adjoint operator** \(H\) on a Hilbert space such that non-trivial zeros of \(\zeta(\tfrac12 + iE) = 0\) correspond to eigenvalues \(E_n\) of \(H\).

| Aspect | Verdict |

| [[berry-keating-hamiltonian]] as proof | **dead** as stated — not self-adjoint |
| HP → flavor / Yukawa | **dead** direct — [[why-not-zeta-flavor-numerology]] |
| HP → primes via spectrum | **watch** — if operator found, primes enter via explicit formula |

## What Failure Would Look Like

- Or: operator exists but **no** connection to any QFT observable — arithmetic stays isolated
