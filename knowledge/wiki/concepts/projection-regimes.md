# Projection Regimes

## Proven (repo labeling + fits)

- Three sector-specific parameter bundles documented in [[manuscript-key-results]] and code paths.
- Parameters **do not** transfer across sectors ([[repo-scientific-findings]]).
- **Quark:** 0% strict survivors (diag 21, PDG-relative tolerances).
- **Charged lepton:** archived 60% range-based (full 3-mass opt); **diag 22 (100 geom, train split):** 1% strict/legacy — train m_μ+m_τ perfect 95%, holdout m_e median loss ~32, 0% within 5%.
- **Neutrino:** archived 45% range-based (480 geom); **diag 23 (100 geom, 90 solved):** 78.9% strict/legacy; g_env ~0.47, not manuscript 0.60.

## Conjecture

- Regimes correspond to different effective information compression ([[information-measure-for-projection-regimes]]).
- \(g_{\text{env}}\) drives neutrino anarchy — manuscript correlation **weak** (\(r < 0.1\)); diag 23 (100 geom): r(g_env, θ₂₃)≈0.07, 95% CI [−0.07, 0.29] — treat as speculative.

## Refuted / overstated

- "Three-regime framework **validated**" for quarks — diagnostics + honest manuscript figures show \(m_c\) systematic failure.
- Regimes as proof of [[information-creates-reality]] without an entropy measure.

## Three Regimes (This Repo)

| Regime | Sector | Dominant effect | Key parameter shift |
|--------|--------|-----------------|-------------------|
| Envelope-dominated | Quarks | Distance suppression | Baseline \(\sigma, k, \eta\) |
| Phase-sensitive | Charged leptons | Interference phase | \(k_e, \eta_e\) differ — diag 22: \|r(k, m_\mu)\| < 0.15; holdout m_e fails |
| Metric-dominated | Neutrinos | Envelope compression | \(g_{\text{env}}\) fit ~0.47 (diag 23, 100 geom); legacy ~0.60 |

## Information Interpretation (Speculative — Refuted for Regime Labels)

Each regime *might* correspond to different information compression — but **`diagnostics/12_regime_entropy_correlation.py` refuted** correlation between regime labels and \(S(\rho_Y)\), effective rank, or off-diagonal entropy (n = 10,080, max \|r\| = 0.047). See [[information-measure-for-projection-regimes]].

[[repo-scientific-findings]]: parameters do not transfer — each regime requires **sector-specific** tuning. Any fundamental theory must explain *why* the projection map differs per sector, with falsifiable predictions.
